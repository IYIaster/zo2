Zo2 Framework

Last updated: May 24th, 2013

Introduction

Zo2 Framework is an open source for building Joomla templates.
Official website: http://zo2framework.org
The source code is licensed under GPL v2, and available at https://github.com/aploss/zo2/

Key Features

  1. Flexible layout, with 12 columns
  2. Reponsive layout based on Twitter Bootstrap
  3. Beautiful Bootstrap typography with Font Awesome Icons
  4. Combine and minify JS & CSS
  5. Shortcode
  6. Mega menu
  7. Theme color
