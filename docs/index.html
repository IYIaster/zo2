Zo2 provides many ready-to-use short codes, it is clear that we cannot cover all kind of short codes for different users. ZO2 makes it simple to make your own shortcodes * The short code structure The first of all we need to do is understand the structure of a short code. 1. Some Shortcodes that we built in the shortcodes folder ( {root}/plugins/system/zo2/shortcodes ): ![integrate shortcode](img/shortcodes.png) 2. This folder includes scripts, styles, images of a shortcode. ( {root}/plugins/system/zo2/addons/shortcodes ): ![integrate shortcode](img/shortcodes-assets.png) * For examples The following image below, the developers can easily create a short code ![integrate shortcode](img/example.png) A example about a short code structure **Notes** (1) - Function name (2) - A Short code tag ### How to include css and javascript only when needed by a shortcode? Zo2 framework provided some functions to do this, the developers can use functions following: - Zo2Framework::addScriptDeclaration - Zo2Framework::addJsScript - Zo2Framework::addCssStylesheet ### How to add a shortcode button in the editor of an article Find "$shortcoders = array(" string then adding new a item inside this array. Copy image of button into the path: {root}/plugins/system/zo2/shortcodes/images .For example, You can see the image below: ![integrate botton toolbar](img/btn-toolbar.png) **Notes** - name: name of shortcode - desc: Description of shortcode. - syntax: syntax of shortcode. - image: icon of shortcode. ### How do I use my shortcode? That is simple, just need to add the shortcode tag in the editor of an article.
